Featured Projects 💡

Here are some of the projects I’ve worked on, focusing on data analysis, machine learning, and real-world applications. Each project involved working with large datasets, applying statistical and predictive modeling techniques, and generating meaningful insights.

📌 Criminal Analysis of Boston & Los Angeles

This project examines crime trends from 2020 to 2024, using geospatial visualization techniques to map crime distributions. Linear regression models were applied to explore relationships between crime rates and socioeconomic factors, helping to identify patterns and trends that could inform public safety strategies.

🎵 Spotify’s Top 50 Tracks Analysis

By analyzing Spotify’s Top 50 tracks, this project aimed to uncover insights into what makes a song popular. Using Python and Pandas, I explored features like danceability, energy levels, and genre classification to understand how song attributes correlate with audience engagement.

🌐 VitalPath (HackHarvard 2024)

Developed as part of HackHarvard 2024, this project focused on building an AI-powered platform that predicts potential health conditions based on user input and symptom patterns. Machine learning models were trained on medical datasets, and the website provided real-time recommendations and remote healthcare connections.

🔧 Methods & Tools Used

  • Programming: Python (Pandas, NumPy, Matplotlib, Scikit-learn), JavaScript (D3.js)
  • Data Analysis & Visualization: Geospatial Mapping, Regression Modeling, Data Preprocessing
  • Machine Learning: Predictive Analytics, Classification Models, Feature Engineering
  • Web Development: HTML, CSS, JavaScript, Flask
